Type Definitions

configuration

The top level configuration object.
Properties:
Name Type Default Description
platform string express Server platform to use. Currently only express is supported,
basePath string . Base path to use for the application,
configFile string azureMobile.js Name of the file that exports configuration objects to load,
promiseConstructor Promise Promise constructor to use,
tableRootPath string /tables Directory to load table configuration from,
debug bool false Run the server in debug mode. Automatically turned on when node is executed with the --debug option,
version string Current version of the Azure Mobile Apps SDK,
logging loggingConfiguration Logging configuration
data dataConfiguration Data configuration
auth authConfiguration Authentication configuration
cors corsConfiguration Cross-origin resource sharing configuration

loggingConfiguration

Logging configuration
Properties:
Name Type Default Description
level string info Minimum level of messages to log
transports Object Console Hash of winston transports to log messages to
See:
  • https://github.com/winstonjs/winston

dataConfiguration

Data configuration. See sqlServerDataConfiguration for details on configuring SQL Server
Properties:
Name Type Default Description
provider string memory Data provider to use. Supported providers are sql and memory

sqlServerDataConfiguration

SQL Server data configuration
Properties:
Name Type Default Description
user string User name to connect with
password string Password for user
server string Hostname of the database server
port integer 1433 Port to connect to
database string Name of the database to connect to
connectionTimeout integer 15000 Connection timeout in milliseconds
options Object Additional options
Properties
Name Type Description
encrypt bool Encrypt the connection. Required and turned on automatically for SQL Azure
See:
  • https://www.npmjs.com/package/mssql

authConfiguration

Authentication configuration
Properties:
Name Type Default Description
secret string Key to use to sign and validate JWT tokens
audience string urn:microsoft:windows-azure:zumo Token audience claim
issuer string urn:microsoft:windows-azure:zumo Token issuer claim
expiresInMinutes integer 1440 Expiry of signed tokens
See:
  • http://jwt.io/
  • https://github.com/auth0/node-jsonwebtoken

corsConfiguration

Cross-origin resource sharing configuration
Properties:
Name Type Default Description
maxAge integer 300 How long the results of a preflight request can be cached in a preflight result cache,
origins Array.<string> localhost Array of allowed origins

context

This object is attached to the express request object as the azureMobile property.
Properties:
Name Type Description
query queryjs/Query The parsed OData query if provided and the parseQuery middleware has been executed
id string | number The ID associated with the request if provided and the parseQuery middleware has been executed
item object The item being inserted or updated if provided and the parseItem middleware has been executed
req express.Request The current express request object
res express.Response The current express response object
data data The configured data provider
tables function A function that accepts a string table name and returns a table access object for the above provider
user auth/user The authenticated user object if the authenticate middleware has been executed
results object The results of the executeOperation middleware

tableDefinition

Specifies options for table access
Properties:
Name Type Default Description
authorise boolean false Execute the authorise middleware for each table operation
autoIncrement boolean false Automatically increment the id column on each insert
dynamicSchema boolean true Dynamically create table schema
name string Name of the table
