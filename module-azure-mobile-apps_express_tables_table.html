Module: azure-mobile-apps/express/tables/table

This module provides functionality for configuring features of individual tables with an Azure Mobile App.

(require("azure-mobile-apps/express/tables/table"))(definition)

Creates an instance of a table configuration helper.
Parameters:
Name Type Description
definition tableDefinition Additional table options. These properties can also be specified directly on the helper object itself.
Returns:
An object with the members described below.

Methods

(inner) use(middleware)

Specify middleware to be executed for every request against the table
Parameters:
Name Type Description
middleware function An array or argument list of middleware to execute. This middleware must contain the middleware exposed through table.execute.

(inner) read(handler)

Register a table read logic handler. The property also exposes a use function for specifying middleware for the read operation. Middleware must contain the middleware exposed through table.operation. You can also set the authorise property on this member.
Parameters:
Name Type Description
handler tableOperationHandler A function containing logic to execute each time a table read is performed.
Example
var table = require('azure-mobile-apps/express').table();
table.read.authorise = true;
table.read.use(customMiddleware, table.operation);
table.read(function (context) {
    // add an additional query operator
    context.query.where({ property: 'value' });
    return context.execute();
});

(inner) update()

Identical syntax and semantics to the read function, but for update operations.

(inner) insert()

Identical syntax and semantics to the read function, but for insert operations.

(inner) delete()

Identical syntax and semantics to the read function, but for delete operations.
